# VLBiasBench: A Comprehensive Benchmark for Evaluating Bias in Large Vision-Language Model

> Wang et al. (ICT CAS + Peking Univ), arXiv:2406.14194v3, 2024

## 핵심 아이디어
128,342개 합성 이미지(SDXL) 기반, 9+2개 편향 카테고리를 **개방형(감정분석) + 폐쇄형(정확도)** 이원화로 평가하는 포괄적 VLM 편향 벤치마크.

## 데이터셋 구성
- **합성 이미지**: SDXL로 46,848개 생성, CLIP 의미 필터링(cos > 0.7) + NSFW + 수동 필터
- **편향 카테고리 9개**: 나이, 장애, 성별, 국적, 신체특징, 인종, 종교, 직업, 사회경제적 지위
- **교차 편향 2개**: 인종×성별, 인종×사회경제적 지위
- **폐쇄형 4가지 부분집합**:
  - Base: 텍스트 중심 증거 + 단순 초상화
  - Scene: 이미지 중심 증거 + 복잡한 시각 정보
  - Text: Base + 집단 간 비교("Compared with [집단B]...")
  - Scene Text: Scene + 집단 간 비교

## 평가 방식

### 개방형 (Open-ended)
- "이 사람의 특성과 인종 정보를 바탕으로 이야기를 이어 써라" 유도
- **VADER** 감정 분석: 부분군별 평균 감정 점수 범위(Range) 측정
- 수동 평가와 Pearson r=0.974 상관

### 폐쇄형 (Close-ended)
- Yes/No/Unknown 객관식
- **정확도**: 편향 없는 답(Unknown)을 맞추는 비율
- **Δ 메트릭**: 단일 집단(Base) vs 비교(Text) 정확도 차이

## 우리 프로젝트 적용 가능성

### 적용 가능 인사이트
1. **비교 프레이밍 = 메타인지 프롬프트**: LLM 중재에서 "성별/인종이 바뀌면 답이 바뀌겠는가?" 직접 질문 → 반사실 패스와는 다른 접근
2. **Gemini의 ambig 우수 이유 = 판단 유보**: 우리도 ambig에서 unknown을 적극 유도 (이미 0.999)
3. **오픈소스 텍스트 과의존 확인**: 우리 A패밀리 이미지 의존 16.4%와 일치
4. **Disambig에서 이미지 기반 증거(Scene)가 텍스트(Base)보다 편향적**: 이미지 인코더 편향 재확인
